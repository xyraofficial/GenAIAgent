# GenAI Agent

## Overview

GenAI Agent is an Android AI assistant application built with Material Design 3 (Material You) principles. The app provides a conversational AI interface with a focus on clean chat experiences, intelligent feedback states, and modern mobile design patterns inspired by Google Messages, ChatGPT mobile, and the Gemini app.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Design Framework
- **Material Design 3**: The application follows Material You guidelines with dynamic theming, elevated surfaces, and fluid interactions
- **Typography**: Roboto Flex font family with a defined scale (Display 32sp, Headline 24sp, Body Large 16sp, Body Medium 14sp, Label 12sp)
- **Spacing System**: 4dp base unit with standard increments (4dp, 8dp, 16dp, 24dp)

### Screen Architecture
1. **Welcome/Onboarding Screen**: First-launch experience with hero illustration, capability showcase, and primary call-to-action
2. **Main Chat Interface**: Three-part structure with app bar (64dp), scrollable conversation area, and expandable input bar (56dp min)

### UI/UX Patterns
- Conversational-first design with clean message hierarchy
- Visual AI thinking states and response streaming feedback
- Adaptive layouts supporting device rotation and keyboard interactions
- Trust indicators for security without exposing technical details

## External Dependencies

### Design Resources
- **Roboto Flex**: Material Design 3 standard font family
- **Material Design 3 Components**: Standard Android Material component library

### Platform
- **Android**: Native Android platform target
- **Material You**: Dynamic theming system integration
